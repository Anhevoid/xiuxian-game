# 一念成仙 — 修仙放置小游戏

纯前端修仙放置游戏，打开即玩，无需安装任何依赖。

> **试玩：** 直接在浏览器打开 `index.html`，或访问 [GitHub Pages](#)。

## ✨ 游戏简介

你将扮演一位修仙者，从练气期起步，通过**修炼 / 打坐 / 突破 / 战斗 / 炼丹**逐步提升实力，历经九大境界，最终**渡劫飞升**。

---

## 🎮 核心玩法

| 系统 | 说明 |
|------|------|
| 🧘 **修炼突破** | 自动修炼获取修为，打坐吐纳加速，修为攒满后可突破境界（小境界高成功率，大境界走九重天劫） |
| ⚔️ **秘境战斗** | 9 大秘境 × 3 档妖兽（弱/中/强），实时回合战斗，掉落灵石/灵草/妖核 |
| 📕 **功法习得** | 6 本功法（吐纳术→鸿蒙造化功），修炼速度依次叠加 |
| ⚔️🛡️ **装备系统** | 武器×9 + 护甲×9，每级战力指数增长（×2.3），旧装回收 |
| 💊 **丹药炼化** | 灵草+妖核炼制回血丹/修为丹/突破丹/抗劫丹，成功率随境界提升 |
| 📿 **法宝炼器** | 6 件法宝可升阶（凡->灵->法->仙->神），镶嵌铭纹，分解材料得铭纹 |
| 🏘️ **洞府建设** | 8 类建筑（灵田/灵泉/聚灵阵/丹房/藏书阁/兽栏/炼器炉/避劫阵），产出+加成+离线产出 |
| 🐾 **灵宠相伴** | 6 种灵宠出战助战+被动加成，升星进化 |
| ✨ **奇遇事件** | 随机触发奇遇，多选项多结果，可触发强制战斗/天劫 |
| 🍀 **气运因果** | 气运影响概率，因果影响天劫/掉落，卜卦消耗气运 |
| 🌟 **三道分支** | 元婴期择仙/魔/佛道途，独立技能树，影响加成/结局 |
| ⚔️ **斗法论道** | 斗法/论道得感悟，点亮悟道树，段位累加 |
| 🔄 **轮回转世** | 飞升/兵解结算道果，跨周目点亮轮回天赋树（6 支 5 级） |
| 🏯 **宗门系统** | 万剑宗/药王谷/天机阁/玄武宗 四大宗门，加入获取属性加成，可付费转宗 |
| 📜 **悬赏任务** | 击杀/打坐/炼丹/突破/积攒五类悬赏，完成领取灵石+贡献，清完自动刷新 |
| 🛒 **贡献商店** | 宗门贡献兑换法宝/专属功法/丹药 |
| 💾 **存档系统** | localStorage 自动存档，离线收益结算（最高 8 小时），v1→v2→v3 温和迁移 |

## 🚀 启动方式

### 方式一：直接打开（推荐）
双击 `index.html`，或在浏览器中打开它即可。

### 方式二：本地服务器
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# VS Code
安装 Live Server 插件，右键 → Open with Live Server
```

### 方式三：GitHub Pages
Fork 本仓库，在 Settings → Pages 中启用，选择 `master` 分支即可。

## 🛠 技术栈

- **纯前端** — HTML + CSS + JavaScript (ES5 strict)，零依赖
- **file:// 可运行** — 不依赖服务器，本地双击即玩
- **localStorage** — 存档/读档/离线收益
- **响应式** — 适配桌面与移动端

## 📝 设计哲学

- **飞升即起点** - v3 引入轮回元进度：飞升结算道果，跨周目点亮轮回天赋树，每一世更强
- **前快后慢不破** - 修为需求增长远快于修炼速度增长；新系统温和乘子叠加，不改 GROW=2.3 战力曲线
- **风险×收益×因果** - 九重天劫多波应对，气运影响概率、因果影响天劫，选择有重量
- **差异化重玩** - 三道分支（仙/魔/佛）× 轮回天赋 × 灵宠/洞府配装，每周目体验不同
- **放置友好** - 离线收益（最高 8 小时，50% 效率），洞府产出纳入离线结算
- **温和迁移** - 存档 v1->v2->v3 升级不清档，旧存档自动补全字段

## 🐛 调试模式

按 `Ctrl+Shift+D` 打开测试面板，提供：
- 修为/灵石快速调整
- 境界自由切换
- 一键突破 / 解锁全部秘境
- 无敌模式

**调试状态不写入存档，刷新页面后复原。**
