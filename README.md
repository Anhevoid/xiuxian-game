# 一念成仙 — 修仙放置小游戏

纯前端修仙放置游戏，打开即玩，无需安装任何依赖。

> **试玩：** 直接在浏览器打开 `index.html`，或访问 [GitHub Pages](#)。

## ✨ 游戏简介

你将扮演一位修仙者，从练气期起步，通过**修炼 / 打坐 / 突破 / 战斗 / 炼丹**逐步提升实力，历经九大境界，最终**渡劫飞升**。

---

## 🎮 核心玩法

| 系统 | 说明 |
|------|------|
| 🧘 **修炼突破** | 自动修炼获取修为，打坐吐纳加速，修为攒满后可突破境界（小境界 90% 成功率，大境界动态难度） |
| ⚔️ **秘境战斗** | 9 大秘境 × 3 档妖兽（弱/中/强），实时回合战斗，掉落灵石/灵草/妖核 |
| 📕 **功法习得** | 6 本功法（吐纳术→鸿蒙造化功），修炼速度依次叠加 |
| ⚔️🛡️ **装备系统** | 武器×9 + 护甲×9，每级战力指数增长（×2.3），旧装回收 |
| 💊 **丹药炼化** | 灵草+妖核炼制回血丹/修为丹/突破丹，成功率随境界提升 |
| 📿 **法宝系统** | 6 件法宝（玄火鉴/金刚镯/嗜血珠/聚宝盆/灵泉玉瓶/造化玉碟），击杀强怪掉落或宗门贡献兑换 |
| 🏯 **宗门系统** | 万剑宗/药王谷/天机阁/玄武宗 四大宗门，加入获取属性加成，可付费转宗 |
| 📜 **悬赏任务** | 击杀/打坐/炼丹/突破/积攒五类悬赏，完成领取灵石+贡献，清完自动刷新 |
| 🛒 **贡献商店** | 宗门贡献兑换法宝/专属功法/丹药 |
| 💾 **存档系统** | localStorage 自动存档，离线收益结算（最高 8 小时），v1→v2 温和迁移 |

```
练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 炼虚期 → 合体期 → 大乘期 → 渡劫期 → 🌅 飞升
```

## 🚀 启动方式

### 方式一：直接打开（推荐）
双击 `index.html`，或在浏览器中打开它即可。

### 方式二：本地服务器
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# VS Code
安装 Live Server 插件，右键 → Open with Live Server
```

### 方式三：GitHub Pages
Fork 本仓库，在 Settings → Pages 中启用，选择 `master` 分支即可。

## 🛠 技术栈

- **纯前端** — HTML + CSS + JavaScript (ES5 strict)，零依赖
- **file:// 可运行** — 不依赖服务器，本地双击即玩
- **localStorage** — 存档/读档/离线收益
- **响应式** — 适配桌面与移动端

---

## 📝 设计哲学

- **前快后慢** — 修为需求增长（1.4×）远快于修炼速度增长（1.16×），越往后越需要功法/法宝/宗门加成
- **风险收益** — 大境界突破成功率低（30%~65%），失败有惩罚（走火入魔）；强怪掉落好但战力要求高
- **放置友好** — 支持离线收益（最高 8 小时，50% 效率），页面隐藏同样结算
- **温和迁移** — 存档版本升级不清档，旧存档自动补全字段

## 🐛 调试模式

按 `Ctrl+Shift+D` 打开测试面板，提供：
- 修为/灵石快速调整
- 境界自由切换
- 一键突破 / 解锁全部秘境
- 无敌模式

**调试状态不写入存档，刷新页面后复原。**
